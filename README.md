# Superstore Sales Dashboard (Power BI)

## Overview
This project is an interactive Sales Dashboard built in Power BI using the Superstore dataset.  
The dashboard provides insights into sales performance, profit, customer segments, product performance, and category-wise sales trends.

## Key Features
- KPI Cards for:
  - Total Sales
  - Total Profit
  - Total Orders
  - Total Customers
- Monthly Sales Trend Analysis
- Profit by Market
- Sales by Segment
- Top 10 Products by Sales
- Sales by Category
- Interactive slicers for:
  - Month
  - Year
  - Market

## Tools & Skills Used
- Power BI
- Power Query (Data Cleaning)
- DAX Measures
- Data Visualization
- Dashboard Design
- Interactive Filtering with Slicers

## Insights Covered
- Sales trends across months
- Profit contribution by market
- Customer segment-wise sales performance
- Best-selling products
- Category-wise sales comparison

## Learning Outcome
This project helped me understand:
- Data cleaning in Power Query
- Creating calculated columns and measures using DAX
- Building interactive dashboards in Power BI
- Choosing appropriate visuals for business analysis
